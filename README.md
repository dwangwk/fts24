# Ecobridge

## Introduction
EcoBridge, is a platform that utilizes smart contracts and custodian wallets to consolidate the tokenised carbon credit markets.

## Tooling
- JavaScript (React.js, Next.js, Ether.js)
- Solidity
- Firebase / Firestore

### Project Structure

```bash
.

```
